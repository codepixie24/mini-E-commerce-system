# mini-E-commerce-system
It is a console based mini ecommerce application built using java to practise and streghthen Object-Oriented programming (oop) concepts.

`Project overview`

This project simulates a simple online shopping system where users can:
*view available products to a shopping cart 
*add products to a shopping cart 
*view cart contents 
*calculate total cost
*practise basic customer-product interactions.
THE PROJECT WAS CREATED AS PART OF OOP EXAM PREP PREPARATION AND JAVA PROGRAMMING PRACTISE.

`Features`

*product management 
*shopping cart system
*menu-driven console interface
*price calculation
*object-oriented design
*use java collections (Arraylist)

`OOP Concepts used`

This project demostrates important Object-Oriented Programming principles :
a. classes and objects
b. encapsulation
c. constructors 
d. methods
e. composition
f. collections 
g. acess modifiers 
h. user interaction
